# SQL Analytics Use Cases

## Overview
This project demonstrates business-focused SQL analytics using a sales dataset.
The queries answer common stakeholder questions using aggregations, window functions,
and performance-aware SQL patterns.

## Concepts Covered
- Aggregations & GROUP BY
- Window functions
- Ranking & top-N analysis
- Time-based analysis
- Revenue contribution analysis

## Files
- schema.sql – table structure
- data.sql – sample data
- queries.sql – analytics queries
- insights.md – business insights
